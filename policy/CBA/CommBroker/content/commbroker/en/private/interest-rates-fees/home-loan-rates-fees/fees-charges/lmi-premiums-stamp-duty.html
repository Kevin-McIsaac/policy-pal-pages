Home Loan Lenders' Mortgage Insurance Premiums and Stamp Duty

 Back to Home Loan Fees and Charges

The LMI premium payable is typically based on the size of the deposit, and the value of the property. The circumstances of the loan will also determine whether LMI may apply..

Refer to Security Lending Margins for LVR limits and use the Postcode Lookup Tool to determine specific LMI Policy.

 

Note 1: The minimum LMI premium for all home loans* is $1,118 (plus stamp duty).

 

Note 2: When a customer has previously paid an LMI premium and any properties securing the new loan also secure any existing Commonwealth Bank Loan(s), our system will automatically calculate and apply the correct LMI premium. Refer to the LMI calculation example page for more information.

 

Note 3: A 15% premium loading applies to applications where the value of the investment property security represents >50% of the total property security value. The loading is added to the LMI calculation. The Bank's system automatically calculates the premium. Refer to LMI calculation example page for more information.

 

Note 4: Geographic premium loadings apply depending on the circumstances of the loan. Refer to the Postcode Lookup Tool to determine if a geographic loading applies. The loading is added to the LMI calculation. The Bank's system automatically calculates the premium. Refer to LMI calculation example page for more information.

Stamp Duty

Stamp duty varies between States and Territories and is thus dependent on the location of the security property.

State or Territory Rate of Duty
NSW/ACT  0.0%
TAS / VIC / NT / WA 10.7317073171% 
SA 11.8048780488%
QLD 9.6585365854%

Where the loan security consists of two or more properties located in different States, the premium will then be apportioned to the relevant state on a security value basis. The Stamp duty will then be calculated on the premium apportioned to each State at the rate applicable. 

Refer to Lender Mortgage Insurance for more information.
