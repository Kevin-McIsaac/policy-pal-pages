To assist with reaching faster decisions and outcomes for you and your customers, we have identified some key areas of opportunity for when you are submitting applications.

 

Application Submission 

Mandatory Documents
  • Completed and signed Home Loan On-boarding form, including mandatory Responsible Lending questions
  • Identification documents
  • Pricing approval confirmation, if applicable
  • Guarantors consent form, if applicable
  • Interest Only Simulator if applicable
Imaging Documents
  • Ensure the Bank Reference Number only is in the email subject line – ensure the BRN is captured correctly and matches to the application - if there are additional characters or numbers in the subject line the documents may not be received causing delays to your application.
  • Acceptable file types - Only the following file types are acceptable: .doc, .docx, .xls, .xlsx, .pdf, .html, .htm, .bmp, .tif, .tiff, .jpeg, .gif, .zip, .ppt, .eml, .txt, .rtf – anything outside of this will not accept and will cause the document file to reject into a separate process that may cause delays in processing and matching.
  • If you have images embedded into your email they will be classified as separate documents, i.e. an image in your signature or aggregator logo are identified as images and created as separate documents. Removal of images is preferred.
  • File size – the current system is limited to 20mb, should the file be too large you will need to break up the transmissions and send multiple emails.
General Information
  • Order the valuation upfront and submit it with application
  • Include a copy of the servicing calculator
  • All required verifications documents to support income 
  • Use CommApply Online Dynamic Supporting Document Checklist – tailored for the application you are submitting.

 

Capturing of application data

All home loan applications are initially assessed through an application scorecard.

The information captured in the application, as well as behavioural information we already hold for existing customers, helps with the assessment and highlights the need to ensure the information you collect and input is accurate.

When submitting applications it is important that all information is input correctly as this will ensure applications are not referred back for missing or more information.
