Example 1: No existing loans

For customers with no existing loans with the Bank, the base loan amount is the total amount of their borrowings including fees financed.

Refer to the following table as a guide to calculating the LDP for customers with no existing loans with the Bank.

We’ll use the following amounts as an example:

  • Loan amount - $90,000
  • Establishment fee - $600.

Stage

Description

1
  • Loan amount to calculate the base LVR $90,600, ie. $90,000 + $600
  • Security value $100,000 (NSW owner occupied)
  • $90,600 / $100,000 x 100 = 90.60% base LVR, refer Loan to valuation ratio (LVR) for details.

2

LDP rate for a loan of $90,600 with an LVR of 90.60% is 1.7518181818% (Premium rates as at 29th July 2023).

3

LDP premium before state-based loading and product discount: $90,600 x 1.7518181818% = $1,587.15.

4​

LDP state-based loading:

$1,587.15 x 0% =$0

$1587.15 + $0 = $1,587.15
​5

Product discount: $1,587.15 x 0.11% = 174.59

6

Total LDP premium payable: $1,587.15 – 174.59 = $1,412.41

Note: LDP rounded down to the nearest dollar $1,412.

7

Total amount insured: $92,012 (90,600 + $1,412), which is shown on the Consumer Credit Contract (CCC) Schedule.

8

Capitalised LVR: $92,012 / $100,000 x 100 = $92.01%

 

Example 2: Existing loan covered by LDP

Refer to the following table as a guide to calculating the LDP for customers with a top up on an existing loan with the Bank.

Original loan (July 2009 rates)​

Top Up (29th July 2023 rates)​
  • Loan amount: $525,000
  • Interest type: Interest Only
  • Security: $630,000
  • LVR: 83.3%
  • LDP rate: 0.0%
  • LDP fee (this figure becomes the LDP credit for future deals): $5,136.
  • Current loan balance: $525,000
  • Top Up amount: $265,000
  • Total debt (including new money): $790,000
  • Security: $940,000 (NSW owner occupied)
  • LVR: 84.04%
  • LDP rate based on $790,000 (the total debt including new money): 1.2206818182% 
  • LDP credit (previously paid fee): $5,136.​

 

Stage

Description

1
  • Loan amount to calculate the base LVR $790,000, ie. $525,000 + $265,000
  • Security value $940,000 (NSW owner occupied)
  • $790,000 / $940,000 x 100 = 84.04% base LVR, refer Loan to valuation ratio (LVR) for details.

2

LDP rate for a loan of $790,000 with an LVR of 84.04% is 1.2206818182%   (Premium rates as at 29 July 2023).

3

LDP premium before LDP credit, state-based loading and product discount: $790,000 x 1.2206818182% = $9,643.39. 

4​

LDP credit: $9.643.39 - $5,136 (LDP credit, refer ‘Note’ below) = $4,507.39

​5

LDP state-based loading:

$4,507.39 x 0% = $0

$4,507.39 + $0 = $4,507.39

​6

Product discount: $4,507.39 x 0.11% = $495.81

​7

Total LDP premium payable: $4,507.39 - $495.81 = $4,011.58

Note: LDP rounded down to the nearest dollar $4,011.

8

Total amount insured: $794,011 ($790,000 + $4,011), which is shown on the Consumer Credit Contract (CCC) Schedule.

9

Capitalised LVR: $794,011 / $940,000 x 100 = 84.47%.

Note: 'LDP Credit' for the Top Up / additional advance can be obtained from the CommSee 'account summary' screen for each loan.

 

Example 3: Existing loan not covered by LDP

For customers with an existing loan with the Bank that does not have LDP cover the base loan amount is the total combined loan amount plus fees financed.

Refer to the following table as a guide to calculating the LDP for customers with an existing loan with the Bank that does not have LDP cover.

We'll use the following amounts as an example:

  • Existing loan amount (not insured) - $40,000
  • New loan amount - $340,000
  • Establishment fee - $600.

Stage

Description

1​
  • Loan amount to calculate the base LVR $380,600, ie. $40,000 + $340,000 + $600
  • Security value $420,000 (NSW owner occupied)
  • $380,600 / $420,000 x 100 = 90.62% base LVR, refer Loan to valuation ratio (LVR) for details.​

2

LDP rate for a loan of $380,600 is 2.2736363636% (Premium rates as at 29th July 2023). 

3

LDP premium before state-based loading and product discount: $380,600 x 2.2736363636% = $8,653.46. 

​4

$8,653.46 x 0.0% = $0.00 (LDP state based loading, aligns to LMI Stamp Duty rate).

5​

$8,653.46 + $0.00 = $8,653.46

​6

​Product discount: $8,653.46 x 0.11% = $951.88

​7

Total LDP premium payable: $8,653.46 - $951.88 = $7,701.58

Note: LDP rounded down to the nearest dollar $7,701. 

8

Total amount insured: $388,301 ($380,600 + $7,701), which is shown on the Consumer Credit Contract (CCC) Schedule.

9

Capitalised LVR: $388,301 / $420,000 x 100 = 92.45%

 

Example 4: Multiple new borrowings against single security (apportioned LDP)

For customers with multiple new borrowings against the same single security property with the Bank, the LDP is calculated on the total debt and then apportioned to each loan.

Refer to the following table as a guide to calculating the LDP for customers with multiple new borrowings against the same single security property (same application).

We'll use the following amounts as an example:

  • Home Loan:
    • Loan amount - $110,000
    • Establishment fee - $600
    • Total = $110,600
  • Investment Home Loan:
    • Loan amount - $90,000
    • Establishment fee - $600
    • Total = $90,600.

​Stage

​Description

1​
  • Loan amount to calculate the base LVR $201,200, ie. $110,000 + $600 + $90,000 + $600
  • Security value $235,000 (WA owner occupied)
  • $201,200 / $235,000 x 100 = 85.62% base LVR, refer Loan to valuation ratio (LVR) for details.

2​

LDP rate for a loan of $201,200 with an LVR of 85.62% is 00.8852272727% (Premium rates as at 29 July 2023).​

​3

 LDP premium before state-based loading and product discount: $201,200 x 0.8852272727% = $1,781.08

4​

LDP state-based loading:

$1,781.08 x 10.7317073171% = $191.14.

$1,781.08 + $191.14 = $1,972.22

5​

Product discount: $1,972.22 x 0.11% = 216.94

​6

​Total LDP premium payable: $1,972.22 – 216.94 = $1,755.28 (appointed to each loan).

7

Home Loan (calculate apportionment of premium):

  • $110,600 / $201,200 = 54.97%
  • LDP premium payable: $1,755.28 x 54.97% = $964.88 (rounded down to the nearest dollar $964) 
  • Total amount insured: $111,564 ($110,600 + $964), which is shown on the Consumer Credit Contract (CCC) Schedule.

​8

Investment Home Loan (calculate apportionment of premium):

  • $90,600 / $201,200 = 45.03% 
  • LDP premium payable: $1,755.28 x 45.03% = $790.40 (rounded down to the nearest dollar $790) 
  • Total amount insured: $91,390 ($90,600 + $790), which is shown on the Consumer Credit Contract (CCC) Schedule. 

9

Total amount insured: $202,954 (Home Loan $11,564 + Investment Loan $91,390). 

​10

Capitalised LVR: $202,954 / $235,000 x 100 = 86.36% (capitalised LVR).

 

Example 5: Investment security >50% where loading applies

A 15% premium loading applies to applications where the value of the investment property security represents >50% of the total property security value. The loading is added to the LDP calculation.

Refer to the following table as a guide to calculating the LDP for customers with investment security >50%.

We’ll use the following amounts as an example:

  • Loan amount - $590,000
  • Establishment fee - $600
  • Owner occupied security - $325,000
  • Investment security - $350,000 (which represents 52% of total security).

​Stage

​​Description

1​
  • Loan amount to calculate the base LVR $590,600, ie. $590,000 + $600
  • Security value $675,000 (located in TAS)
  • $590,600 / $675,000 x 100 = 87.50% base LVR, refer Loan to valuation ratio (LVR) for details.​​​

​2

LDP rate for a loan of $590,600 with an LVR of 87.50% is 1.2859090909% (Premium rates as at 29 July 2023).​​​​

​3

LDP premium before loadings and product discount: $590,600 x 1.2859090909% = $7,594.58. 

​4

Add 15% investment security loading: $7,594.58 x 115% = $8,733.77.

 

​5

LDP state-based loading:

$8,733.77x 10.7317073171% = $937.28.

$8,733.77 + $937.28 = $9,671.05

​6

Product discount: $9,671.05 x 0.11% = $1,063.32

7

Total LDP premium payable: $9,671.05 - $1,063.32 = $8,607.23

Note: LDP rounded down to the nearest dollar $8,607.23.

8

Total amount insured: $599,207 ($590,600 + $8,607), which is shown on the Consumer Credit Contract (CCC) Schedule.

9

Capitalised LVR: $599,207 / $675,000 x 100 = 87.77%
