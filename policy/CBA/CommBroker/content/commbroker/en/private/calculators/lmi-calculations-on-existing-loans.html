Example 1: No existing loans

For customers with no existing loans with the Bank, the base loan amount is the total amount of their borrowings including fees financed.

Refer to the following table as a guide to calculating the LMI premium for customers with no existing loans with the Bank.

  • Loan amount - $90,000
  • Establishment fee - $600.

​Stage

Description​

1​
  • Loan amount to calculate the base LVR $90,600, ie. $90,000 + $600
  • Security value $100,000 (QLD owner occupied)
  • $90,600 / $100,000 x 100 = 90.60% base LVR, refer Loan to valuation ratio (LVR) for details.​

​2
  • LMI premium rate for a loan of $90,600 with an LVR of 90.60% is 1.7518181818% (Premium rates as at 29th July 2023).​

​3
  • LMI premium before Stamp Duty: $90,600 x 1.7518181818% = $1587.15.

​4
  • Stamp Duty rates on premium: $1,587.15 x 9.6585365854% (Obtained from Stamp Duty rates on premiums table) =$153.30.​

​5

Total LMI premium payable: $1587.15 + $153.30 = $1,740.45 

Note: LMI rounded down to the nearest dollar $1,740.00 

​6

Total amount insured: $92,340 ($90,600 + $1,740), which is shown on the Consumer Credit Contract (CCC) Schedule

​7

Capitalised LVR: $92,340 / $100,000 x 100 = 92.34%

Example 3: Existing loan not covered by LMI

For customers with an existing loan with the Bank that does not have LMI cover the base loan amount is the total combined loan amount plus fees financed.

Refer to the following table as a guide to calculating the LMI premium for customers with an existing loan with the Bank that does not have LMI cover.

  • Existing loan amount (not insured) - $40,000
  • New loan amount - $340,000
  • Establishment fee - $600.

​Stage

​Description

1​
  • Loan amount to calculate the base LVR $380,600, ie. $40,000 + $340,000 + $600
  • Security value $420,000 (QLD owner occupied)
  • $380,600 / $420,000 x 100 = 90.62% base LVR, refer Loan to valuation ratio (LVR) for details.​

​2

LMI premium rate for a loan of $380,600 is 2.2736363636% (Premium rates as at 29 July 2023).​

​3

LMI premium before Stamp Duty: $380,600 x 2.2736363636% = $8,653.46.

​4

Stamp Duty rates on premium: $8,653.46 x 9.6585365854% (obtained from the Bank's Stamp Duty rates on premiums table). = $835.80​

​5

Total LMI premium payable: $8,653.46 + $835.80 = $9,489.26

Note: LMI rounded down to the nearest dollar $9,489.

​6

Total amount insured: $390,089 ($380,600 + $9,489), which is shown on either the Consumer Credit Contract (CCC) Schedule or Notice of Variation (NOV) whichever is relevant.

​7

Capitalised LVR: $390,089 / $420,000 x 100 = 92.88%

 

Example 4: Multiple new borrowings against single security (apportioned LMI)

For customers with multiple new borrowings against the same security property with the Bank, LMI is calculated on the total debt and then apportioned to each loan.

Refer to the following table as a guide to calculating the LMI premium for customers with multiple borrowings against the same security property (same application). 

  • Home Loan:
    • Loan amount - $110,000
    • Establishment fee - $600
    • Total = $110,600
  • Investment Home Loan:
    • Loan amount - $90,000
    • Establishment fee - $600
    • Total = $90,600.

​Stage

​Description

1​
  • Loan amount to calculate the base LVR $201,200, ie. $90,000 + $110,000 + $1,200
  • Security value $235,000 (QLD owner occupied)
  • $201,200 / $235,000 x 100 = 85.62% base LVR, refer Loan to valuation ratio (LVR) for details.​

2​

​LMI premium rate for a loan of $201,200 with an LVR of 85.62% is 0.8852272727% (Premium rates as at 29 July 2023).​​

3​

LMI premium before Stamp Duty: $201,200 x 0.8852272727% = $1,781.08.

4​

Stamp Duty rates on premium: $1,781.08 x 9.6585365854% (obtained from the Bank's Stamp Duty rates on premiums table). = $1172.03.​

5​

$1,781.08 + $172.03 = $1,953.11

Note: LMI rounded down to the nearest dollar $1,953 and then apportioned to each loan.

6​

Home Loan (calculate apportionment of premium):

  • $110,600 / $201,200 = 54.97%
  • LMI premium payable: $1,953 x 54.97% = $1,073.56 (Rounded down to the nearest dollar $1,073)
  • Total amount insured: $111,673($110,600 + $1,073) which is shown on the Consumer Credit Contract (CCC) Schedule

7​

Investment Home Loan (calculate apportionment of premium):

  • $90,600 / $201,200 = 45.03%
  • LMI premium payable: $1,953 x 45.03% = $879.44 (Rounded down to the nearest dollar $879)
  • Total amount insured: $91,479 ($90,600 + $879), which is shown on the Consumer Credit Contract (CCC) Schedule

8​

Total amount insured: $203,152 ( Home Loan $111,673 + Investment Home Loan $91,479), which is shown on the Consumer Credit Contract (CCC) Schedule.

​9

Capitalised LVR: $203,152 / $235,000 x 100 = 86.45%

 

Example 5: Investment security >50% where loading applies

A 15% premium loading applies to applications where investment property security represents >50% of the total security value. The loading is calculated and added to the current LMI calculation.

Refer to the following table as a guide to calculating the LMI premium for customers with investment security >50%.

  • Loan amount - $590,000
  • Establishment fee - $600
  • Owner occupied security - $325,000
  • Investment security - $350,000 (which represents 52% of total security).

​Stage

​​Description

1​
  • Loan amount to calculate the base LVR $590,600, ie. $590,000 + $600
  • Security value $675,000 (located in QLD)
  • $590,600 / $675,000 x 100 = 87.50% base LVR, refer Loan to valuation ratio (LVR) for details.​​

2​

LMI premium rate for a loan of $590,600 with an LVR of 87.50% is 1.2859090909% (Premium rates as at 29 July 2023).​​​

3​

LMI premium before loading and Stamp Duty: $590,600 x 1.2859090909% = $7,594.58

4​

To add 15% investment security loading:

$7,594.58 x 115% = $8,733.77

5​

Stamp Duty rates on premium: $8,733.77 x 9.6585365854% (obtained from the Bank's Stamp Duty rates on premiums table) = $843.55.​​

6​

Total LMI premium payable: $8,733.77 + $843.55 = $9,577.32

Note: LMI rounded down to the nearest dollar $9,577.

7​

Total amount insured: $600,177 ($590,600 + $9,577), which is shown on the Consumer Credit Contract (CCC) Schedule.

8​

Capitalised LVR: $600,177 / $675,000 x 100 = 88.90%

 


Example 6: Geographic LMI policy - where LMI applies from a lower LVR and a loading applies

Refer to the Postcode Lookup Tool for specific LMI policy.

For applications within specific geographic locations LMI can apply from a lower LVR and a geographic loading may apply. The loading is calculated and added to the current LMI calculation.

In the example below the Postcode Lookup Tool quotes the maximum LVR with no LMI is 70% and the Geographic LMI Policy 1 loading applies (a 10% loading is used in this example).

Refer to the following table as a guide to calculating the LMI premium for customers in specific geographic locations. 

  • Loan amount - $500,000
  • Establishment fee - $600
  • Owner occupied security - $680,000

1 
  • Loan amount to calculate the base LVR $500,600, ie. $500,000 + $600
  • Security value $680,000 (located in QLD)
  • $500,600 / $680,000 x 100 = 73.62% base LVR, refer Loan to valuation ratio (LVR) for details.
  • Note: Postcode Lookup Tool quotes LMI is payable above 70% and the Geographic LMI Policy 1 loading applies (in this example a 10% loading applies).

4  To add 15% loading:
 $7,594.58 x 115% = $8,733.77
