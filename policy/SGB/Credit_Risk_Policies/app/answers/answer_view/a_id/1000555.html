1. About this policy
 

1.1 Why this policy?

This policy will help you determine the maximum loan to value ratio (LVR) when assessing a loan application.

You must read this policy in conjunction with Assessing Acceptable Securities policy. The Assessing Acceptable Securities policy will help you identify the types of securities we accept in a consumer mortgage application.

 

1.2 What is LVR?

LVR is a term used to express the percentage ratio of a loan to the value of an asset or assets that provide security for a loan. 

The bank uses LVR to:
  • Assign a maximum percentage we will lend against security property offered, with or without mortgage insurance
  • Show the amount of lending (expressed as a percentage ratio) a customer may already have against a property
  • Identify when Lender’s Mortgage Insurance (LMI) is required.
By using the assigned LVRs in this policy and the security value of the property, we can determine the lending value of the offered security property.

For definition relating to LVR, refer to Glossary for further details.

For information on calculation of LVR, refer to LVR calculation and Exceeding maximum LVR > 18.1 LVR Calculation for further details

If the loan application requires mortgage insurance, refer to  Lenders mortgage insurance (LMI) where required.

2.8 Assessing second mortgage

The bank takes a second mortgage over a security property, where there is a prior mortgagee (e.g. where the first mortgage is held by another financial institution)
Within SGB, BSA, BoM: Westpac and RAMS are considered to be other financial institutions.

Assessing second mortgage lending value

Category Action
Calculation A 20% buffer for contingencies is to be used in the calculation of the Lending value.
a.   Determine the lending value of the offered security property (using the security value and assigned LVR)
b.   Identify the value of the debts secured by the first mortgage with other financial institution (higher of limits or outstanding balance of loan)
c.         Multiply figure at b by 120%
d.         Remaining available lending value = a - c
refer to Example for illustration.
Special condition for settlement Write to the prior mortgagee seeking consent requesting that the prior mortgagee establish their priority amount.
The priority amount must be no more than the amount initially used to calculate the lending value for our loan (i.e. balance outstanding plus 20%). If this occurs, you must seek additional approvals from Credit.
 

2.9 Debt allocation (Debt apportioning)

Debt allocation may occur during a new loan activity (new origination or variation) where the borrower:
  • Has an existing debt secured by two or more properties and the existing debt is not being refinanced (either partially or in full) and
  • Not all properties securing the existing debt will be used as supporting security in the new loan.
Note: Debt allocation is not required where all real estate assest are offered as supporting security in the new loan application / variation.

The following debt allocation principles are applied by the Bank:
  • For properties offered as security, the security value used in the debt allocation tool must be the same as the security value used in the assessment of the new loan activity. For all real estate assets that are not offered as security in the new loan activity the latest recorded valuation figure is used as the market value.
  • Ensure that each property has the maximum permitted non-MI LVR loaded in the debt allocation tool.
  • If two or more applications are proceeding at the same time, assessment notes must identify which application will settle / be drawn down first. Ensure the new debt of the first application is shown as a debt allocation in the second and subsequent application(s).
  • If two or more loans will settle simultaneously then the applications must mirror each other. Eg All applications must reflect the new debt of the other.
  • Debt allocation is not available for mortgage insured applications unless all insured security is linked to the new loan (this will however result in the LMI premium needing to be based on the total secured OOT and will require referral to Mortgage insurers for a manual premium calculation)
  • Part debt allocation can only occur between properties owned by the borrower. The only exception is where the borrower is a company and the sole security is offered by the company directors
  • Part debt allocation cannot occur to a non-borrower’s property, for example where security guarantee has been provided.  

Example of an acceptable debt allocation scenario
The applicants want to borrow to purchase an investment property and need to use their owner occupier property to cover the shortfall in security for the investment loan. They previously provided a Family Pledge security guarantee to their children (they were the guarantor).
E.g. The applicants previously provided a limited guarantee of $100k to their Child as a Family Pledge Security Guarantee using their owner occupier property
Any existing Family Pledge security guarantee must be included for debt allocation purposes.
When calculating the LVR for new investment home loan, we will need to allow for the existing $100k family pledge security guarantee.  

Example of an unacceptable debt allocation scenario
The applicants want to purchase an investment property and need to use their owner occupier security to cover the shortfall in security for the investment loan.
The applicants used the Family Pledge Home Loan option to help them purchase their owner occupier property.  E.g. their parents provided a limited family pledge security guarantee of $200k to help them buy their owner occupier property. The borrower’s debt cannot be allocated to the guarantee property.
For the transaction to be able to proceed, the applicants may need to restructure the existing owner occupier debt to release the family pledge security guarantee.
There are limited scenarios where the applicant may increase their borrowings where a loans is supported by an existing Family Pledge security guarantee, for further information refer to  Guarantees > Family Pledge Home Loan Option.
